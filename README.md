# Amazon Sales Analysis

## Project Overview
This project analyzes Amazon sales data to identify trends, customer behavior, product performance, and geographical distribution to support data-driven business decisions.

## Objectives
- Analyze overall sales performance and trends
- Identify popular product categories
- Understand customer buying behavior
- Provide actionable business recommendations

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Repository Structure
- data/ : Dataset used for analysis
- code/ : Python notebooks
- reports/ : Final analysis report
- outputs/ : Charts and visualizations

## Key Outcomes
- Sales trend analysis over time
- Category-wise sales insights
- Actionable recommendations for business improvement
